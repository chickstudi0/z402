# Example: Express with LibSQL / Turso Persistence

This example uses the `z402-storage-libsql` storage backend.

- Suitable for edge deployments (Cloudflare, Vercel Edge, Fly.io, etc.)
- Shared global low-latency data layer
- Fully HTTP-based database connectivity

Choose this if you want:
- Edge-speed paywalled APIs
- Geo-distributed access control
- Minimal infra overhead

Run:
```ts
npm start
```
